# 2026-03-20 Repo Bootstrap

## Task

Create a dedicated repository for AI agent coding methodology, management, and retrospectives.

## Outcome

Worked:

- established a clear repository scope
- created a stable directory structure
- added rule, workflow, template, research, and retrospective entry points

Failed or incomplete:

- no maturity model yet
- no evaluation framework yet
- no tool-specific comparison matrix yet

## Failure Surface

The first risk was over-building the repository before defining its operating model.

## Missing Signals

- a clear initial content boundary
- a promotion path from research to stable rules

## Root Cause

- vague repository shape at project start

## Fix

- define the repository around rules, playbooks, templates, research, and retrospectives
- keep the first version intentionally small

## Promotion Candidate

- new repositories about agent methodology should start with both a rule system and a retrospective system
