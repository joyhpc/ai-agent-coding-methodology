# Execution Plan Template

## Goal

State the desired end state in one paragraph.

## Constraints

State non-negotiable constraints.

## Assumptions

List assumptions that could change the plan.

## Plan Steps

1. inspection
2. implementation slice 1
3. verification
4. implementation slice 2
5. final verification

## Verification Strategy

List the exact checks and pass criteria.

## Risks and Rollback

- main risks
- rollback trigger
- rollback path

## Progress Log

Update this during execution.

## Decision Log

Record important decisions and why they were made.

## Retrospective Notes

Record lessons that should become rules, templates, or playbooks.
