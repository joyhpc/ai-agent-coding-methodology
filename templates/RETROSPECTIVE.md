# Retrospective Template

## Task

Name and summarize the task.

## Outcome

What worked and what failed?

## Failure Surface

Where did the agent or workflow break down?

## Missing Signals

What signal, instruction, test, or artifact was missing?

## Root Cause

Choose the dominant cause:

- vague task
- missing rule
- missing verification
- poor decomposition
- environment gap
- tool limitation
- model limitation

## Fix

What should change now?

- new repo rule
- better template
- better playbook
- better verification script
- task decomposition change

## Promotion Candidate

What should be promoted into a stable rule?
